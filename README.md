# GeoPhone

GeoPhone - простая программа для Mac, которая позволяет выбрать точку на карте и временно отправить эту геолокацию на подключенный iPhone.

## Скачать

Актуальная версия: **GeoPhone v0.1.23**

[Скачать GeoPhone_iPhone_v0.1.23_build24_2026-07-10.zip](https://github.com/Marfys9/GeoPhone-Releases/releases/download/v0.1.23/GeoPhone_iPhone_v0.1.23_build24_2026-07-10.zip)

## Что внутри архива

- `GeoPhone.dmg` - установочный файл для Mac.
- `INSTRUCTION_RU.txt` - инструкция на русском языке.

## Как начать

1. Скачайте ZIP по ссылке выше.
2. Распакуйте архив.
3. Откройте `INSTRUCTION_RU.txt` и выполните шаги из инструкции.
4. Установите GeoPhone через `GeoPhone.dmg`.
5. При первом запуске пройдите мастер настройки в приложении.

## Требования

- Mac с актуальной macOS.
- iPhone с включенным режимом разработчика.
- USB-кабель для подключения iPhone к Mac.
- Интернет на этапе первой настройки, чтобы GeoPhone мог скачать необходимые служебные файлы.

## Важно

- Перед сменой геолокации лучше закрыть лишние приложения на iPhone.
- Подключите iPhone кабелем, разблокируйте его и нажмите «Доверять этому компьютеру», если телефон попросит.
- Возврат к реальной геолокации обычно происходит самостоятельно через некоторое время, после сброса в GeoPhone или после перезагрузки iPhone.
- Во время настройки macOS может несколько раз запросить пароль администратора. Это нужно для установки и запуска служебных компонентов, которые помогают Mac работать с iPhone.
